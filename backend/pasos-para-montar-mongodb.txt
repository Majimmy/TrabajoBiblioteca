montar mongo data base:
1.instalar docker
2.hacerlo funcionar
3.ir a terminal en la carpeta backend y escribir:
   docker run --name mongodb -d -p 27017:27017 mongo
----------(docker debe estar encendido)-----------
4.iniciar la instancia mongodb

visualizar los datos de el nuevo container mongodb:
5.instalar mongo compass
6.utilizar la ruta de la base de datos default
(mongodb://localhost:27017)
7.ver la base de datos "test"
